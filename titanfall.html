Titanfall 2 UI Redesign

UX Case Study UI Design Figma Game UI

Overview

A self-initiated UX/UI case study in which I researched and redesigned several screens from Titanfall 2's user interface. The project applies modern UX principles — clarity, hierarchy, accessibility — to an existing shipped game UI as a design exercise and portfolio piece.

The project was published on ArtStation in December 2024.

Process

  • Conducted a heuristic evaluation of Titanfall 2's existing menus and HUD elements
  • Identified key usability issues: cluttered information hierarchy, low contrast text on busy backgrounds, and inconsistent iconography
  • Created redesigned screen mockups in Figma addressing each identified issue
  • Documented the before/after rationale for each design decision

Tools Used

  • Figma (mockups & component design)
  • Heuristic evaluation methodology
  • ArtStation (publication & presentation)
