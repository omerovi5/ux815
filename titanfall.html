Titanfall 2 UI Redesign

UX Case Study UI Design Figma Game UI ArtStation

Overview

This is a self-initiated UX/UI case study in which I researched, evaluated, and redesigned several screens from Titanfall 2's user interface. Published on ArtStation in December 2024, the project applies modern UX principles to a shipped AAA game title as a design exercise and portfolio piece.

Titanfall 2 is a critically acclaimed first-person shooter developed by Respawn Entertainment. Despite its strong gameplay, several areas of its UI presented usability and accessibility opportunities that I found worth exploring.

Research & Evaluation

Before touching Figma, I spent time playing through the game with a critical UX lens, documenting pain points across the main menu, loadout screen, and in-game HUD. I then conducted a formal heuristic evaluation against Nielsen's 10 usability heuristics.

  • Visibility of system status — The HUD lacked clear indicators for wall-run stamina and Titan build progress during hectic gameplay moments
  • Aesthetic and minimalist design — The loadout screen displayed a high volume of information simultaneously with no clear visual hierarchy
  • Consistency and standards — Icon language varied inconsistently between the campaign and multiplayer sections
  • Accessibility — Several text elements across menus fell below minimum contrast ratios for readability

Design Decisions

Each redesigned screen was accompanied by written rationale documenting what changed and why. Key decisions included consolidating the loadout screen into a progressive disclosure model that shows essential options first and reveals advanced settings on demand, and rebuilding the HUD to use a radial layout that keeps critical info at the edges of the player's natural focal point rather than in the corners.

I deliberately stayed close to Titanfall 2's existing visual language — dark, industrial, high-tech — rather than reinventing it. The goal was to improve usability within the established aesthetic, not to redesign the game's identity.

Tools Used

  • Figma (mockups & component design)
  • Heuristic evaluation (Nielsen's 10)
  • WCAG contrast analysis
  • ArtStation (publication)
