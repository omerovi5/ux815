Backyard Brawl

Party Game Unity Multiplayer UI/UX

Overview

Backyard Brawl is a fast, fun party game featuring two distinct game modes designed for local multiplayer:

  • Candy Collectathon — A trick-or-treating royale set on Halloween night. Players knock on houses to collect candy and race to deposit it into the witch's cauldron to score points before other players steal it.
  • LaserBall — An arcade-style soccer match where players score goals using laser-powered shots and wild power-ups.

My Role

I led the UI/UX design and implementation for the entire game — from the main menu and mode selection screens to in-game HUDs for both modes. Each mode needed its own visual language while still feeling cohesive as a single product, which was the core design challenge I solved through consistent typography and a shared colour system.

Tools & Tech

  • Unity 6 / C#
  • Figma (UI design & prototyping)
  • itch.io deployment
  • Cross-discipline team collaboration
