Volley-Guns

Indie Game Unity UI/UX Team Project

Overview

Volley-Guns is an independent video game about playing volleyball with various firearms. Players serve, spike, and block using an arsenal of weapons — each with unique physics and trajectory properties that change how the volleyball behaves mid-air.

The game was developed collaboratively with my fellow peers from my Game Design Minor cohort at Michigan State University over the course of more than a year, and is soon to be released on Steam.

My Role

As the UI/UX designer on the team, I was responsible for designing and implementing all in-game menus, HUD elements, and the main menu experience. I created wireframes and high-fidelity mockups in Figma before bringing them to life inside Unity 6 using C# and Unity's UI Toolkit.

Tools & Tech

  • Unity 6 / C#
  • Figma (wireframing & UI design)
  • Git / version control
  • Team collaboration across disciplines
